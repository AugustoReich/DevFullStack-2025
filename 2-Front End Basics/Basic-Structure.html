Kawarimi

O Kawarimi no Jutsu (Técnica de Substituição Corporal) é um ninjutsu básico ensinado na Academia Ninja, que permite ao usuário substituir o seu próprio corpo por um objeto próximo, como um tronco de madeira, no exato momento em que é atacado. O objetivo é criar uma ilusão visual, <header> enganando o oponente para que pense que o ataque acertou o ninja, o que dá tempo para ele fugir ou contra-atacar.

Clone das Sombras

O jutsu Clone das Sombras, introduzido por Tobirama Senju no anime e mangá Naruto, cria clones físicos do usuário, ao contrário dos clones ilusórios. Subscrito Para isso, o chakra do usuário é distribuído igualmente entre os clones, o que permite que eles ataquem e executem outras técnicas, mas também os torna vulneráveis, pois se dispersam com um golpe. A técnica é considerada proibida (kinjutsu) por seu alto consumo de chakra, naruto que pode esgotar o usuário e levá-lo à morte, especialmente se for utilizado para criar múltiplos clones.
